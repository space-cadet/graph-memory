# T9: Temporal Decay + Relationship Strength

**Status:** ⏳ pending  
**Priority:** 🟡 MEDIUM  
**Started:** 2026-06-19  
**Phase:** 4 of 6

## Goal

Distinguish active relationships from stale ones.

## Problem

All 3,653 relationships are treated equally. Can't answer "what am I actively working on?"

## Solution

```sql
ALTER TABLE relationships ADD COLUMN strength REAL DEFAULT 1.0;
ALTER TABLE relationships ADD COLUMN half_life_days INTEGER DEFAULT 30;
```

**Decay formula:**
```
strength(t) = strength_0 × 0.5^(days_elapsed / half_life)
```

**Default half-lives:**
| Relationship Type | Half-life |
|-------------------|-----------|
| mentioned_with | 7 days |
| uses | 30 days |
| created | 90 days (no decay) |
| decided_to_use | 180 days |

**Reinforcement:** +0.2 per mention, max 1.0
