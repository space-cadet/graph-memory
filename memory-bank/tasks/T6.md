# T6: LLM-Based Entity Extraction

**Status:** ⏳ pending  
**Priority:** 🔴 HIGH  
**Started:** 2026-06-19  
**Phase:** 1 of 6

## Goal

Replace regex-based entity extraction with LLM-based extraction to capture concepts, decisions, topics, and questions that regex currently misses.

## Problem Statement

Current regex patterns in `session-entity-extractor.cjs`:
- Catch: files (`*.ts`), tools (`git`, `npm`), explicit `[[links]]`
- Miss: "quantum computing", "chimera-chat" (plain text), "we decided to use Chart.js"

Test results on this machine:
- "quantum" → 0 results
- "chimera" → 0 results
- "obsidian" → 0 results

Only 154 entities from 931 sessions — most are files and tools.

## Solution

### Extraction Prompt Template

```
Analyze this conversation and extract structured information:

1. ENTITIES: People, projects, concepts, tools, files, papers mentioned
   Format: [{"name": "...", "type": "project|concept|person|tool|file|paper", "confidence": 0.0-1.0}]

2. DECISIONS: Explicit choices made
   Format: [{"what": "...", "rationale": "...", "alternatives": ["..."]}]

3. TOPICS: Discussion subjects
   Format: ["...", "..."]

4. QUESTIONS: Unresolved questions
   Format: ["...", "..."]

5. RELATIONSHIPS: Meaningful connections
   Format: [{"source": "...", "target": "...", "type": "uses|depends_on|implements|decided_to_use", "strength": 0.0-1.0}]

Conversation:
{conversation_text}
```

### Implementation Plan

1. Create `scripts/llm-extractor.cjs`
2. Support batch processing (10-50 sessions per call)
3. Add confidence scoring
4. Store results in extended schema

### Schema Changes

```sql
ALTER TABLE entities ADD COLUMN confidence REAL DEFAULT 1.0;
ALTER TABLE entities ADD COLUMN description TEXT;
ALTER TABLE relationships ADD COLUMN confidence REAL DEFAULT 1.0;
ALTER TABLE relationships ADD COLUMN strength REAL DEFAULT 1.0;
ALTER TABLE relationships ADD COLUMN context TEXT;
```

### Cost Analysis

| Model | Cost/1K tokens | Avg session tokens | Cost/session | 13K sessions |
|-------|---------------|-------------------|-------------|-------------|
| GPT-4o-mini | $0.00015 | ~2K | $0.0003 | ~$4 |
| GPT-4o | $0.005 | ~2K | $0.01 | ~$130 |
| Local (Ollama 7B) | $0 | ~2K | $0 | $0 |

**Recommendation:** Start with GPT-4o-mini for speed, migrate to local model for cost.
