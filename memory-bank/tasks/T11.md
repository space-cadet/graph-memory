## Goal

Integrate graph queries into the agent memory pipeline so I use the graph during conversations.

## Problem

I never query the graph. I read MEMORY.md and session files instead.

## Solution

```javascript
// In agent memory retrieval
async function retrieveMemory(query) {
  const graphResults = await queryGraph(query, { minConfidence: 0.7 });
  if (graphResults.length > 0 && graphResults[0].confidence > 0.7) {
    return formatGraphResults(graphResults);
  }
  return readMemoryMd(); // fallback
}
```

**Requirements:**
- Query latency < 100ms
- Confidence scoring
- Graceful degradation
